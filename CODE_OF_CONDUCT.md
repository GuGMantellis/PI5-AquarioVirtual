# Código de Conduta (Contributor Covenant)

Como contribuidores e mantenedores deste projeto, nós nos comprometemos a respeitar todas as pessoas que contribuem através de issues, pull requests ou quaisquer outros canais de comunicação.

Esperamos que todos que participam do projeto demonstrem respeito, empatia e colaborem de forma construtiva. Assédio de qualquer tipo não será tolerado.

## Nossos Padrões

Exemplos de comportamento que contribuem para criar um ambiente positivo incluem:
* O uso de linguagem acolhedora e inclusiva.
* Respeito a pontos de vista e experiências diferentes.
* Aceitar críticas construtivas com graciosidade.
* Focar no que é melhor para a comunidade e para o projeto.
* Mostrar empatia para com outros membros.

Exemplos de comportamento inaceitável por participantes incluem:
* O uso de linguagem ou imagens sexualizadas, bem como atenção sexual não desejada.
* Comentários insultuosos ou depreciativos (trolling) e ataques pessoais ou políticos.
* Assédio público ou privado.

Se você observar algum comportamento inaceitável, por favor, reporte aos mantenedores do projeto.
