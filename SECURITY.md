# Política de Segurança

## Versões Suportadas

Apenas a versão mais recente do software e firmware recebe atualizações de segurança.

| Versão | Suportado          |
| -------| ------------------ |
| v1.0.x | :white_check_mark: |
| < 1.0  | :x:                |

## Reportando uma Vulnerabilidade

Por favor, não abra issues públicas para problemas de segurança.
Envie um e-mail para a equipe de desenvolvimento. Nós tentaremos responder em até 48 horas e fornecer um cronograma estimado para a correção.
