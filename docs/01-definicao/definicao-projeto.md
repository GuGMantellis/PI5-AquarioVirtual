# Definição do Projeto

**Título:** Aquário Virtual Inteligente (Projeto Integrador 5)

## Visão Geral
Este projeto tem como objetivo desenvolver um sistema completo para o monitoramento e gerenciamento de aquários. Através do uso de sensores IoT, atuadores e uma interface web/mobile, o usuário pode acompanhar em tempo real as condições da água e os equipamentos do aquário.

## Objetivos
1. **Monitoramento:** Coletar dados de sensores (temperatura, pH, nível da água).
2. **Controle:** Acionar atuadores como relés para iluminação, bombas e alimentadores automáticos.
3. **Interface:** Prover um Dashboard intuitivo para análise dos dados.

## Público-Alvo
Hobbyistas, profissionais de aquarismo e pesquisadores que necessitam de um controle rigoroso sobre os parâmetros ambientais de seus aquários.
