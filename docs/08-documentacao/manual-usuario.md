# Manual do Usuário

Bem-vindo ao Aquário Virtual!

## Acesso ao Dashboard
1. Acesse o endereço IP local do seu projeto ou a URL hospedada (ex: `http://localhost:3000`).
2. A tela inicial mostrará os indicadores principais: **Temperatura**, **pH** e **Nível da Água**.

## Controles
- **Luz:** Use o interruptor na interface para ligar e desligar a iluminação do aquário.
- **Alimentação:** Pressione "Alimentar Agora" para acionar o motor de passo do alimentador.

## Alertas
- Se a temperatura ultrapassar 28°C ou ficar abaixo de 24°C, um alerta visual aparecerá na tela.
