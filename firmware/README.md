# Firmware - Aquário Virtual

Este diretório contém o código fonte para o microcontrolador responsável por fazer a leitura dos sensores e acionamento dos atuadores.

## Dependências
- Biblioteca para Sensor de Temperatura (DallasTemperature / OneWire)
- Biblioteca WiFi
- Biblioteca FirebaseClient (se aplicável)

## Como compilar
Abra o projeto na IDE do Arduino ou use PlatformIO. Selecione a placa correta (ex: ESP32 Dev Module) e clique em compilar/upload.
